## Installation

To run locally you will need [NodeJS](https://nodejs.org/) and [Yarn](https://yarnpkg.com).

```shell
git clone git@github.com:itzsaga/slack-list.git
cd slack-list
yarn
yarn start
```

This will run a local server and you can browse the site.
